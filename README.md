# OpenClaw Pinokio Installer

One-click installer for **OpenClaw** (formerly ClawdBot) via Pinokio.

## Features
- Installs the latest version of OpenClaw
- Easy setup
- One-click launch

## How to Install
1. Open Pinokio
2. Go to **Discover** → **Community** tab
3. Search for "OpenClaw"
4. Click Install
